# MyWatchlist
This is an application which will help in making your own watch-list of movies.
# Major Component
- Kotlin
- Android SDK 33
- Material 3
- MVVM with Repository
- Dagger Hilt
- Navigation Component(fragments)
- Lottie
- Coil
- Coroutines
- Retrofit
- Room
